# Summary

This is my first big 7 segment clock based on Wemos D1 Mini Pro + Neopixels. It is flat so it can be hanged on the wall. It uses my WiFi network to retrieve the time and the temp and Watts so it is pretty accurate. :-)
It is ready to print. 
Use natural filament for the segments (crystals) and whatever color for the body and central control box.
Hope you like it.
GITHUB code: https://github.com/Menyiques/-Seven_segment_Arduino_Clock
